# Features

## What can Next.js do ?

- **Advanced rendering methods**
  - Server-side rendering (SSR)
  - Static site generation (SSG)
  - Incremental static generation (ISR)
  - Partial pre-rendering
  - HTML Streaming
- **Routing**
  - File-system based router
  - Client-side navigation

- **Default CSS Support**

  - Images
  - Fonts
