# What are server actions ?

Asynchronous functions executed on the server <br/>A way of executing code on the server without making an explicit fetch call <br/>
React feature (not Next.js specific)

**Where?**

- Can be called in server components
- Can be called in client components

**For what?**

- To handle form submissions
- To handle data mutation
